# MCMC Trace File Note

## File: `mcmc_trace.nc`

**Size**: 1.9 GB
**Format**: NetCDF (Network Common Data Format)
**Status**: Not tracked in Git (too large for repository)

### Contents

This file contains the complete MCMC trace for the Hierarchical Bayesian Spatial Model:
- **Chains**: 2
- **Draws per chain**: 2,000
- **Total samples**: 4,000
- **Parameters**: 67,786
  - Fixed effects (β): 3
  - SA4 random effects: 107
  - SA3 random effects: 358
  - SA2 random effects: 2,472
  - SA1 spatial effects (φ): 61,844
  - Variance components: 5
  - Spatial autocorrelation (ρ): 1

### How to Access

The trace file can be regenerated by running:
```bash
python3 hierarchical_bayesian_spatial_model.py
```

Runtime: ~8-10 minutes

### Loading the Trace

```python
import arviz as az

# Load trace
trace = az.from_netcdf('bayesian_spatial_results/mcmc_trace.nc')

# View summary
print(az.summary(trace))

# Access specific parameters
beta_samples = trace.posterior['β']
rho_samples = trace.posterior['ρ']
spatial_effects = trace.posterior['φ']
```
